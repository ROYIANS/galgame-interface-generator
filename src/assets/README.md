# Assets 文件夹

此文件夹用于存放 **需要被 Vite 构建处理** 的静态资源（如图片、字体等）。

## ⚠️ 背景音乐文件位置

**背景音乐 `bgm.mp3` 应该放在 `public/` 文件夹，而不是这里！**

因为音频文件较大，放在 `public/` 目录可以：
- 避免被构建工具处理（更快的构建速度）
- 确保文件在打包后直接复制到 dist 目录
- 在开发和生产环境都使用相同路径

## 正确的文件位置

```
项目根目录/
├── public/
│   └── bgm.mp3          ← 背景音乐放这里！
├── src/
│   └── assets/
│       └── (图片等其他资源)
```

## 如何添加背景音乐

1. 将音乐文件重命名为 `bgm.mp3`
2. 放到项目根目录下的 `public/` 文件夹
3. 重新构建：`npm run build`
4. 音乐会自动出现在 `dist/bgm.mp3`

更多说明请查看 `public/README.md`
